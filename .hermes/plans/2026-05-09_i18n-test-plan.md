# 易观 i18n 翻译功能 — 测试计划

目标: http://49.235.108.61:8080

## 1. 冒烟测试（API 层）

| # | 用例 | 方法 | 预期 |
|---|------|------|------|
| 1.1 | 健康检查 | GET /health | 200 |
| 1.2 | 未登录访问翻译接口 | GET /api/history/1/translate?target=en | 401 |
| 1.3 | 登录获取 token | POST /api/auth/login | 200 + token |
| 1.4 | 查询不存在的翻译 | GET /api/history/99999/translate?target=en | 404 |
| 1.5 | 无效 target 参数 | GET /api/history/1/translate?target=fr | 400 |

## 2. 核心翻译流程

| # | 用例 | 方法 | 预期 |
|---|------|------|------|
| 2.1 | 获取用户历史记录 | GET /api/history | 200，每条记录含 lang 字段 |
| 2.2 | GET 已有翻译缓存 | GET /api/history/{id}/translate?target=en | 200 + cached:true |
| 2.3 | POST 生成新翻译 | POST /api/history/{id}/translate?target=en | 200 + cached:false |
| 2.4 | 幂等：再次 POST 同一翻译 | POST 同上 | 200，不重复调用 LLM |
| 2.5 | 原文与目标语言相同 | GET /api/history/{id}/translate?target=zh (原文zh) | 200 + source:"original" |

## 3. 前端 UI 测试（需浏览器）

| # | 用例 | 预期 |
|---|------|------|
| 3.1 | 登录页 tab 显示 | "登录"/"Login" 正常，不显示原始 key |
| 3.2 | 切换语言后历史记录 | 展开非当前语言的解读，显示翻译提示条 |
| 3.3 | 点击翻译按钮 | 显示 loading → 替换为翻译文本 |
| 3.4 | 切回原语言 | 恢复原始文本，不再调 API |
