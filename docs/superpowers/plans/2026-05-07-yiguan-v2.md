### 任务 1：数据库抽象层 — Store 接口 + SQLite 建表

**文件：**
- 创建：`internal/store/store.go`
- 创建：`internal/store/sqlite/sqlite.go`

**Store 接口定义：**

```go
package store

import "time"

type User struct {
    ID        int64     `json:"id"`
    Phone     string    `json:"phone"`
    Nickname  string    `json:"nickname"`
    Avatar    string    `json:"avatar"`
    Address   string    `json:"address,omitempty"`
    Password  string    `json:"-"` // bcrypt hash, never expose
    CreatedAt time.Time `json:"created_at"`
}

type Quota struct {
    ID        int64     `json:"id"`
    UserID    int64     `json:"user_id"`
    QuotaType string    `json:"quota_type"` // "free", "paid", "share", "ad"
    CreatedAt time.Time `json:"created_at"`
    UsedAt    *time.Time `json:"used_at,omitempty"`
}

type History struct {
    ID             int64     `json:"id"`
    UserID         int64     `json:"user_id"`
    Question       string    `json:"question"`
    PrimaryGua     string    `json:"primary_gua"`
    ChangingGua    string    `json:"changing_gua"`
    YaoPositions   string    `json:"yao_positions"`
    Interpretation string    `json:"interpretation"`
    CreatedAt      time.Time `json:"created_at"`
}

type Store interface {
    // 用户
    CreateUser(phone, password, nickname string) (*User, error)
    GetUserByPhone(phone string) (*User, error)
    GetUserByID(id int64) (*User, error)
    UpdateUser(id int64, nickname, address string) error

    // Quota
    GetRemainingQuota(userID int64) (int, error)
    AddQuota(userID int64, quotaType string) error
    ConsumeQuota(userID int64) error  // 扣减 1 次

    // 历史记录
    SaveHistory(h *History) error
    GetHistory(userID int64, limit, offset int) ([]History, error)

    // 管理
    GetTotalUsers() (int64, error)
    ListUsers(limit, offset int) ([]User, error)

    Close() error
}
```

**SQLite 建表 DDL：**

```sql
CREATE TABLE IF NOT EXISTS users (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    phone TEXT UNIQUE NOT NULL,
    nickname TEXT NOT NULL,
    avatar TEXT DEFAULT '',
    address TEXT DEFAULT '',
    password TEXT NOT NULL,
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP
);

CREATE TABLE IF NOT EXISTS quotas (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    user_id INTEGER NOT NULL,
    quota_type TEXT NOT NULL DEFAULT 'free',
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    used_at DATETIME,
    FOREIGN KEY (user_id) REFERENCES users(id)
);

CREATE TABLE IF NOT EXISTS history (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    user_id INTEGER NOT NULL,
    question TEXT NOT NULL,
    primary_gua TEXT NOT NULL,
    changing_gua TEXT NOT NULL,
    yao_positions TEXT NOT NULL,
    interpretation TEXT NOT NULL,
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (user_id) REFERENCES users(id)
);
```

**JWT 生成：**

```go
import "github.com/golang-jwt/jwt/v5"

func generateToken(userID int64, secret string) (string, error) {
    claims := jwt.MapClaims{
        "user_id": userID,
        "exp":     time.Now().Add(15 * time.Minute).Unix(),
        "iat":     time.Now().Unix(),
    }
    token := jwt.NewWithClaims(jwt.SigningMethodHS256, claims)
    return token.SignedString([]byte(secret))
}
```

```go
func AuthRequired(secret string) func(http.Handler) http.Handler {
    return func(next http.Handler) http.Handler {
        return http.HandlerFunc(func(w http.ResponseWriter, r *http.Request) {
            tokenStr := extractBearer(r)
            if tokenStr == "" {
                http.Error(w, `{"error":"需要登录"}`, 401)
                return
            }
            claims, err := parseJWT(tokenStr, secret)
            if err != nil {
                http.Error(w, `{"error":"token无效"}`, 401)
                return
            }
            userID := int64(claims["user_id"].(float64))
            ctx := context.WithValue(r.Context(), "user_id", userID)
            next.ServeHTTP(w, r.WithContext(ctx))
        })
    }
}
```

## 环境变量

| 变量 | 说明 | 必需 |
|------|------|------|
| `DASHSCOPE_API_KEY` | 千问 API Key | 是 |
| `JWT_SECRET` | JWT 签名密钥 | 是（生产） |
| `YIGUAN_THEME` | 默认主题 light/dark | 否 |

## 启动方式

```bash
# 后端
cd yiguan
export DASHSCOPE_API_KEY="your-key"
export JWT_SECRET="your-secret"
go run cmd/server/main.go

# 前台
cd web/front && npm run dev

# 后台
cd web/admin && npm run dev
```
