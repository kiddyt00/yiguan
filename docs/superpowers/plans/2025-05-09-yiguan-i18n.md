# 中英文切换 (i18n) 实现计划

> **面向 AI 代理的工作者：** 必需子技能：使用 superpowers:subagent-driven-development（推荐）或 superpowers:executing-plans 逐任务实现此计划。步骤使用复选框（`- [ ]`）语法来跟踪进度。

**目标：** 全站支持中英文切换，包括所有前端 UI 文案和 AI 解读内容。

**架构：** 轻量级自研 i18n 方案（不引入 vue-i18n 依赖）。创建 `useI18n()` composable + 双语言字典。语言偏好存 localStorage。NavBar 添加语言切换按钮。后端 LLM prompt 根据语言参数动态生成英文或中文 prompt。

**技术栈：** Vue 3 Composition API, Pinia, Go (后端 prompt 模板)

const lang = ref(localStorage.getItem('lang') || 'zh')

export function useI18n() {
  const setLang = (l) => {
    lang.value = l
    localStorage.setItem('lang', l)
  }

  const t = (key) => {
    const dict = lang.value === 'en' ? en : zh
    return dict[key] || dict[key] || key
  }

  return { lang: computed(() => lang.value), setLang, t }
}
