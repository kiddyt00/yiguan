# 易观·AI 算卦 — 产品需求文档

> 版本: v1.0 MVP | 平台: Web | 更新: 2026-05-07

---

## 一、产品目标

提供首次免费在线算卦服务，通过 **AI 解卦 + 大师私聊转化（私域流量）+ 广告** 实现变现。

## 二、核心流程

```
用户打开 → 输入问题 → 点击"开始提问"
    ↓
系统生成本卦、变卦、变爻 → 调用通义千问 API → 展示 AI 解卦结果
    ↓
底部展示"周易大师一对一详解"按钮 → 点击弹出微信二维码
    ↓
页内植入广告位（首页 + 结果页）
```

## 三、MVP 一期功能清单

| 模块 | 功能 | 说明 |
|------|------|------|
| 问题输入页 | 输入框 + 提交按钮 | 用户输入想问的问题 |
| 算卦引擎 | 铜钱起卦算法 | 6爻，自下而上，3枚铜钱 |
| 卦象展示 | 本卦 + 变卦 + 变爻 | 可视化六爻 |
| AI 解卦 | 调用千问 API | 固定 prompt 模板 |
| 解卦结果页 | 含分享按钮 | 展示完整解读 |
| 大师咨询入口 | 弹出微信二维码 | 周易大师微信 |
| 广告位 | 首页 + 结果页 | 预留广告位 |

## 四、商业模式

- **首次免费** — 流量钩子
- **转发裂变** — 转发给 2 个不同用户获得免费额度（二期）
- **广告收入** — CPC/CPM
- **大师导流** — 合作分成

## 五、算卦逻辑

### 起爻算法

每次抛 3 枚铜钱：正面=3，反面=2，每枚正反概率相同
单次总和 ∈ {6,7,8,9}

- 6 = 老阴（阴→阳，可变）
- 7 = 少阳（阳，不变）
- 8 = 少阴（阴，不变）
- 9 = 老阳（阳→阴，可变）

6 次结果自下而上依次对应：初爻 → 上爻

### 变卦生成

**方法一（6/9 变爻）：** 所有值为 6 或 9 的爻发生阴阳变化

**方法二（55 减总数定主变爻）：**
```
总数 = 6 个爻数值之和
余数 = (55 - 总数) % 6，若余数为 0 则取 6

数爻路径（循环往复）:
1→2→3→4→5→6→6→5→4→3→2→1→1→2→...

从初爻开始，按路径数"余数"步，落点为主变爻位
主变爻强制变化（即使非 6/9，且与 6/9 去重）
```

### 输出

- 本卦（六爻）
- 变卦（变更后的卦）
- 变爻位置（标注所有变化的爻，主变爻特别标识）

## 六、技术要点

| 层 | 选型 | 说明 |
|----|------|------|
| 前端 | HTML + HTMX + Tailwind CSS | 无框架依赖，轻量 SPA |
| 后端 | Go | 算卦引擎 + 千问 API 转发 |
| AI | 通义千问 API | 固定 prompt 模板 |

### Prompt 模板

```
请以专业易经解卦角度，结合问题「{{q}}」和卦象：
本卦：{{b}}，变卦：{{v}}，变爻：{{y}}，给出解读。
请包含：1. 本卦解义 2. 变爻启示 3. 变卦趋势 4. 综合建议
```
