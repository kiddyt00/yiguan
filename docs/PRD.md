# 易观·AI 算卦 — 产品需求文档

> 版本: v2.0 | 平台: Web | 更新: 2026-05-07

---

## 一、产品目标

提供前 3 次免费在线算卦服务，通过 **AI 解卦 + 大师私聊转化（私域流量）+ 广告** 实现变现。

## 二、核心流程

```
用户打开 → 输入问题 → 点击"开始提问"
    ↓
系统生成本卦、变卦、变爻 → 调用大模型 API → 展示 AI 解卦结果
    ↓
底部展示"周易大师一对一详解"按钮 → 点击弹出微信二维码
    ↓
页内植入广告位（首页 + 结果页）
```

## 四、商业模式

- **前 3 次免费** — 流量钩子
- **第 4 次起付费** — A站 9.9 元/次，B站 68 元/次
- **转发裂变** — 转发给 2 个不同用户获得 +1 quota
- **广告收入** — CPC/CPM + 看广告解锁 quota
- **大师导流** — 合作分成

### 双版本策略

| 属性 | A站（平价版） | B站（高价版） |
|------|-------------|-------------|
| 风格 | 浅色系，中国风 | 深色系，科技风 |
| 单价 | 9.9 元/quota | 68 元/quota |
| 免费 quota | 3 次 | 3 次 |
| 转发裂变 | 2人 → +1 | 2人 → +1 |
| 广告解锁 | 30秒 → +1（日限3次） | 30秒 → +1（日限3次） |

## 五、算卦逻辑

（同 v1.0，无变化）

起爻：3枚铜钱×6次 → {6,7,8,9}
变卦：6/9自动变 + 55减总数定主变爻（去重合并）
输出：本卦、变卦、变爻位置（主变爻★标记）

## 六、技术要点

| 层 | 选型 | 说明 |
|----|------|------|
| 前端 | HTML + HTMX + Tailwind CSS | 双主题切换（浅色/深色） |
| 后端 | Go | 算卦引擎 + 大模型转发 + 用户系统 |
| 数据库 | SQLite（MVP） → PostgreSQL | 用户、quota、历史记录 |
| AI | 可配置多模型 | 千问/DeepSeek/GPT 等 OpenAI 兼容接口 |
| i18n | 模板级翻译 | 接翻译大模型批量翻译模板 |

## 七、监管规避策略

1. **形式包装** — 包装为"AI 传统文化学习工具"、"解压互动测试"
2. **境外部署** — 服务器设在香港/新加坡，CDN 加速，域名跳转
3. **支付隐蔽** — 虚拟商品兑换、知识付费、打赏形式收费
4. **内容碎片化** — 短视频+图文问答分发到抖音/小红书
5. **主体模糊** — 注册为文化传媒/科技公司，个人账户收款
